# ⚔️ Albion Online Profit Calculator

Kalkulator zysku z craftowania i rafinacji surowców w grze **Albion Online**.

🔗 **[Otwórz kalkulator](https://twoja-nazwa.github.io/albion-calc/)**

---

## Funkcje

- **⚗ Przerabianie** — Kalkulator rafinacji T2–T8, wszystkie surowce, enchant .0–.4
- **🔨 Craftowanie** — Masowe crafty z dziennikami, focusem i porównaniem z resellerem
- **🔄 Reseller** — Porównanie craft vs skupienie z rynku i sprzedaż na Black Market
- **💰 Skup itemów** — Kalkulator skupu z 3 miejsc × 7 kategorii
- **📥 Eksport XLSX** — 4 arkusze: % zysku, zysk/szt, ceny, wymagane surowce

## Mechaniki

- Iteracyjny zwrot surowców (RRR) z symulacją kaskadową
- Skupienie (Focus) z Tablicą Przeznaczenia + tryb hybrydowy
- Dzienniki rzemieślnicze (Crafting Journals) z kaskadowaniem sławy i arbitrażem PPF
- Opłata stacji (Station Fee / Nutrition)
- Podatek rynkowy Premium/bez + zlecenia kupna/sprzedaży
- Artefakty i Serca peleryn miejskich
- Porównanie Craft vs Resell vs Sprzedaż surowców

## Technologia

- 100% Vanilla JavaScript — brak frameworków, brak npm
- Single-file HTML — działa offline po pobraniu
- localStorage — persystencja ustawień Focus między sesjami
- SheetJS — eksport do XLSX

## Aktualizacje

Wrzuć nowy `index.html` do repozytorium — GitHub Pages zaktualizuje się automatycznie w ciągu ~1 minuty.
